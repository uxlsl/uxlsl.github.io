# 读Python参考手册感想

+ yield 可以有返回值，这点估计很少人知道和使用

+ 常用python文件表示头

```

#!/usr/bin/env python
#-*- coding: utf-8-*-

```

+ exec 执行语句,没返回值, eval 执行表达式，有返回值,它们都可以传入globals(), locals()

```

eval('1+1')
exec('for i in range(100):print(i)')

````


+ 抽象基类的注册, 以后可能有大用
