# spyder 编辑器的使用笔记
spyder 是科学计算的python的使用环境

## 安装
碰到问题
如果ipython 4.0以上,spyder将无法启动,报import 不到指定的模块
解决方法:
将ipython降级为3.2.3以下就可以了

如果在ubuntu环境下,使用官方源安装是比较简单的
sudo apt-get install spyder


## 总结

object inspect, variable explorer是亮点,编辑器只能满足基本要求,
相关配置基本齐全,
