# 终端实现自动更新一行信息

```

import sys
import time
import datetime

sys.stdout.write("now \n")
for i in range(100):
    sys.stdout.write("\r")
    sys.stdout.write(datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S"))
    sys.stdout.flush()
    time.sleep(1)


```
